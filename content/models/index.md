# Models

One space per model. Each space is a folder holding a `model.md` card plus the model's core artifacts — whitepaper, user guide, monitoring plan — so everything about a model lives in one place.

- [CRE PD Model v2.0](/models/cre-pd-v2/model.md) — commercial real estate probability of default. *(sample space)*
- [VA Hedging Model v1.3](/models/va-hedge-v1/model.md) — variable annuity dynamic hedging. *(sample space)*

Whitepapers produced by the docx pipeline (`knowledge/docs/`) attach to their space automatically when the `doc_id` matches the folder's `model_id`.
