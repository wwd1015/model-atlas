# Who to ask

> Names below are role placeholders — replace with your org's real routing when adopting this bundle.

| Topic | First stop | Escalation |
|-------|-----------|------------|
| Model methodology questions | Model owner (on each model card) | Team lead |
| Monitoring runs & thresholds | Monitoring analyst on rotation | Model owner |
| NPI / data handling | Data steward | Compliance officer |
| GenAI tool usage | Team lead | Compliance officer |
| Access requests (data, platforms) | IT service desk | Data steward |
| Atlas content fixes | Anyone — open a PR | Atlas maintainer |
| Model risk / validation process | MRM liaison | Head of MRM |

## Forums

- **Daily standup** — blockers, quick questions.
- **Weekly model forum** — methodology discussions, lesson-learned presentations.
- **Monthly governance review** — change approvals, monitoring exceptions.

If you don't know where a question goes, ask it in the team channel and tag the team lead — misrouted questions are cheap, unasked ones are not. See also [[first-90-days]].
